# 🚀 Full Stack Web Development Journey

Welcome to my learning repository! This workspace is dedicated to tracking my progress, storing exercises, and documenting the projects I build as I work through the Complete Web Development Bootcamp.

## 🎯 Purpose
The goal of this repository is to serve as a personal log and code archive. By documenting my journey, I aim to build a strong foundation in frontend, backend, database management, and Web3 technologies.

## 🛠️ Tech Stack & Skills
Throughout this course, I will be working with:
* **Frontend:** HTML5, CSS3, Flexbox, Grid, Bootstrap, JavaScript (ES6+), DOM Manipulation, jQuery, React.js
* **Backend:** Node.js, Express.js, EJS, RESTful APIs, Unix Command Line
* **Databases:** SQL, PostgreSQL
* **Security:** Authentication, Security, Password Hashing
* **Web3 & Blockchain:** Decentralised Apps (DApps), ICP Live Blockchain, Crypto Tokens, NFTs
* **Tools:** Git, GitHub, Version Control
